# 计算机网络 复习笔记
本仓库整理并汇总了《计算机网络》复习资料，基于谢希仁教材与相关课程与笔记，面向考研复习与自学使用。

**主要目标**
- **定位**: 提供系统化、精简且易查的计算机网络知识点复习材料。
- **受众**: 考研学生、自学者、需要快速回顾网络基础的工程师。

**项目结构**
- **概论/**: 概念与章节笔记（例如 [概论/第二章物理层.md](第二章：物理层.md)）。
- **复习章/**: 对照表与汇总（例如 [复习章/计算机网络对照表.md](复习章/计算机网络对照表.md)）。

**使用方法**
- **阅读笔记**: 直接在本仓库浏览对应 Markdown 文件阅读。
- **查找主题**: 使用编辑器全局搜索关键词（如“信噪比”、“香农公式”）以快速定位相关条目。

**贡献指南**
- **修订/补充**: 欢迎提交 PR 来修正或补充内容，请保持条目简洁、准确并注明参考来源。
- **格式要求**: 使用中文 Markdown，章节标题请以一级/二级标题组织，尽量添加示例或公式说明。

**致谢与来源**
- 教材与课程：谢希仁等教材、王道等教学资源（如适用请在相应笔记中注明参考）。

如需我帮你根据特定章节生成更详细的复习提纲或习题解析，告诉我章节即可。
